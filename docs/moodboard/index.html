Key Visual DNA (what we MUST replicate)

  • Radial color beams — 4-6 saturated stripes radiating from center (THE Tamagotchi signature)
  • PNJ mascots in corners — 4-6 characters with thick black outlines, huge eyes, simple shapes
  • Chrome/metallic text — gradient fills on brand names and CTAs
  • Polka dot pattern backgrounds — large semi-transparent dots on solid colors
  • 25-40 elements per composition — stars, hearts, sparkles, rainbows as visual confetti
  • Bold condensed all-caps type — bright pink or black with colored stroke
  • Round bubbly character shapes — Lisa Frank mascot style with rainbow highlights
  • Never minimal, never fade to white — every pixel is saturated and working
