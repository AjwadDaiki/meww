## Protocole d'absorption (pour Claude Code)

Pour chaque URL de cette liste :

1. **Scraper avec Firecrawl MCP** (format: markdown + images)
2. **Télécharger les 3-5 images clés** de chaque source
3. **Stocker localement** dans `/docs/moodboard/[source-name]/`
4. **Annoter** ce qui t'a marqué dans un fichier `.md` à côté
5. **Synthétiser** les patterns visuels récurrents

Après avoir parcouru les 50 refs, tu dois pouvoir répondre à :

- Quelle palette domine ?
- Quelle typo revient le plus ?
- Comment les PNJ sont typiquement stylisés ?
- Quel niveau de densité visuelle ?
- Comment le texte interagit avec les images ?

**Sans cette absorption, ne commence PAS à coder.** Le rendu sera forcément insuffisant.
