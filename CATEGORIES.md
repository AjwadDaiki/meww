## SEO landing pages par catégorie

Chaque catégorie a sa page `meowreel.com/scene/[slug]` qui cible un keyword précis.

**Exemple pour `midnight-porch-musician` :**
- Title : `AI Cat Porch Video Generator — Your Cat, The Viral Midnight Musician`
- H1 : `Your cat, the midnight porch musician.`
- Description ciblée pour : "ai cat playing instruments", "cat porch video ai", "midnight cat musician"
- Le flow est identique à la landing (upload → buy) mais cette catégorie est pré-sélectionnée

**Impact SEO attendu :** une trentaine de landing pages qui indexent chacune sur long tail → 200-500 visiteurs organiques/jour après 3 mois, gratuit.

## Prompt engineering per category

Chaque catégorie dans MongoDB a :
- `basePrompt` : template avec placeholders
- `negativePrompt` : ce qu'on veut éviter ("no text, no watermark, no human face, no blurry")
- `audioPrompt` : description du son (si modèle supporte, ex: Seedance 2.0)
- `cameraStyle` : "doorbell camera", "cinematic wide", "close-up portrait", etc.

**Exemple complet catégorie `jazz-trumpet` :**

```json
{
  "slug": "jazz-trumpet",
  "basePrompt": "A photorealistic cat wearing a small black suit and bow tie, standing upright on a jazz club stage, playing a golden trumpet with focused expression. The cat is based on the reference image provided. Dim warm stage lights, vintage microphone, hazy smoke atmosphere, empty wooden chairs visible in the background. The cat moves its head slightly to the rhythm. Cinematic 35mm film look, shallow depth of field, warm gold tones.",
  "negativePrompt": "no humans in the scene, no text, no watermark, no logo, not blurry, no distorted anatomy",
  "audioPrompt": "smooth jazz trumpet solo with light cymbal brushes, intimate club ambience",
  "durationSeconds": 5,
  "resolution": "720p",
  "cameraStyle": "cinematic wide shot, warm golden hour feel"
}
```

## Content moderation

Filtrer dès la config :
- Pas de scènes sexualisées (même humoristiques)
- Pas de scènes violentes excessives
- Pas de célébrités nommément citées (risque IP)
- Pas de marques déposées dans les prompts (pas de "Nike", "Apple", etc.)

Pour les catégories style "samouraï" ou "gladiateur" qui impliquent des armes : on garde les armes décoratives (pas d'action de combat).
